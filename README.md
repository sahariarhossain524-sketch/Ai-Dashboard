## 🚀 Features

- **Interactive Analytics**: Real-time data visualization charts built with Recharts.
- **Dark Mode Support**: Seamless light/dark theme switching integrated throughout the UI.
- **Premium Components**: Utilizes modern UI elements and iconography via Lucide React.
- **Responsive Layout**: Sidebar navigation and dashboard grids that adapt perfectly to any screen size.
- **Performance Optimized**: Fast loading speeds and efficient rendering via Next.js App Router.

## 💻 Tech Stack

| Technology | Description |
|------------|-------------|
| **Next.js** | Core React framework and fast server rendering |
| **React** | Component-driven UI development |
| **Tailwind CSS** | Rapid, utility-first styling |
| **Recharts** | Composable charting library |
| **Lucide React** | Beautiful and consistent iconography |

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/sahariarhossain524-sketch/Ai-Dashboard.git
   ```

2. **Navigate to the directory**
   ```bash
   cd Ai-Dashboard
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
